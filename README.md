## 1) Export your data from Meta
- Go to https://accountscenter.instagram.com/info_and_permissions/dyi/
- Choose your Instagram account, choose **Followers and following** ONLY in **Customise information**, **Date range** to **All time**, **Format** to **JSON**, **Media quality** to **Medium quality** (or any), then start export.

## 2) Prepare the workspace
- Put downloaded zip inside the `data` folder.

## 3) Run the script
From the project root:
```bash
python analyze_instagram.py
```
